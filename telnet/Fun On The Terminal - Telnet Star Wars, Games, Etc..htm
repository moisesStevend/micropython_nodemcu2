 █                                                                            ▐▌
 █ Connect to other servers through telnet to view their animated ASCII art,  ▐▌
 █ games, etc they offer the public. If no port is specified than it is the   ▐▌
 █ default port 23; you don't need to specify it.                             ▐▌
 █                                                                            ▐▌
 █ To quit the telnet session: Ctrl + ] (^])                                  ▐▌
 █ To exit telnet type in: quit                                               ▐▌
 █                                                                            ▐▌
 █ Window users do this through the cmd prompt window: Start menu / Run /     ▐▌
 █ type in: cmd                                                               ▐▌
 █ Press OK. Then copy/paste press the telnet cmd line into the cmd prompt    ▐▌
 █ window, press your Enter key to connect.                                   ▐▌
 █                                                                            ▐▌
 █ Linux and Mac users copy/paste the telnet cmd line into your terminal,     ▐▌
 █ press your Enter key to connect.                                           ▐▌
